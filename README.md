# SWAN

# USAGE
create new project
```
swan new NameProject
```

The result

```
└── test
  ├── README.md
  ├── app
  │   ├── controllers
  │   ├── middleware
  │   ├── models
  │   └── views
  ├── config
  │   └── database.yaml
  ├── static
  │   ├── css
  │   ├── images
  │   └── js
  └── vendor
```
